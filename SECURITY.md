# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in Test Generator, please email security@giofrac.com instead of using the issue tracker.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Impact assessment
- Suggested fix (if available)

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.0   | ✅ Yes    |

## Known Security Considerations

- This tool parses Java source files using regex patterns
- Generated test code should be reviewed before running in production
- No sensitive data is collected or transmitted

## Updates

Security updates will be released promptly when vulnerabilities are identified and fixed.
