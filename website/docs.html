Getting Started

Installation

Test Generator is available in multiple ways:

Option 1: Download JAR (Recommended)

# Download the latest JAR from GitHub releases
wget https://github.com/giofrac/test-generator/releases/download/v2.0.0/test-generator-2.0.0.jar

# Or build from source
git clone https://github.com/giofrac/test-generator.git
cd test-generator
./gradlew build

Option 2: Maven Dependency

<dependency>
    <groupId>com.giofrac</groupId>
    <artifactId>test-generator</artifactId>
    <version>2.0.0</version>
</dependency>

CLI Usage

Basic Usage

# Generate tests for all methods in a class
java -cp test-generator-2.0.0.jar com.giofrac.TestGeneratorCLI Calculator.java

# Generate test for a specific method
java -cp test-generator-2.0.0.jar com.giofrac.TestGeneratorCLI Calculator.java --method add

# Specify output directory
java -cp test-generator-2.0.0.jar com.giofrac.TestGeneratorCLI Calculator.java --output src/test/java

Advanced Options

Option Description Example
--method <name> Generate test for specific method --method calculateSum
--output <dir> Output directory for tests --output target/test-gen
--framework <type> Test framework (junit|testng) --framework testng
--mockito Generate tests with Mockito --mockito
--parametrized Generate parameterized tests --parametrized

IntelliJ IDEA Plugin

Note: The IntelliJ plugin requires the IntelliJ Platform SDK and is available in Professional and Enterprise plans.

Installation

  1. Download the plugin JAR from your account dashboard
  2. Open IntelliJ IDEA
  3. Go to File → Settings → Plugins
  4. Click Install Plugin from Disk
  5. Select the downloaded JAR file
  6. Restart IntelliJ IDEA

Usage

  1. Open a Java file with your methods
  2. Place cursor on any public method
  3. Right-click and select Test Generator → Generate Test
  4. The test file will be created automatically in the correct test directory

Advanced Features

Mockito Integration

Generate tests with automatic dependency mocking:

java -cp test-generator.jar com.giofrac.TestGeneratorCLI Service.java --mockito

Parameterized Tests

Create data-driven tests automatically:

java -cp test-generator.jar com.giofrac.TestGeneratorCLI Utils.java --parametrized

TestNG Support

Use TestNG instead of JUnit:

java -cp test-generator.jar com.giofrac.TestGeneratorCLI Calculator.java --framework testng

API Reference

StandaloneTestGenerator

Method Description Parameters
generateTestForMethod() Generate basic JUnit test methodName, returnType, paramTypes, paramNames
generateTestWithMockito() Generate test with Mockito methodName, returnType, paramTypes, paramNames, mockableParams
generateParameterizedTest() Generate parameterized test methodName, returnType, paramTypes, paramNames
generateTestNGTest() Generate TestNG test methodName, returnType, paramTypes, paramNames

Troubleshooting

Common Issues

Ensure your methods are public and properly formatted. The tool only processes public methods.

Make sure you have the IntelliJ Platform SDK configured and the plugin is properly installed.

Check that all required dependencies (JUnit, Mockito, etc.) are in your classpath.
